Install Toast — first launch

Toast is Developer ID signed and Apple-notarized, so macOS should
open it normally after you copy it to Applications.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1) Drag Toast into Applications
   (do this from the DMG window)

2) Open Toast
   • Open Applications
   • Double-click Toast
   • If macOS still warns on an older download, right-click Toast → Open → Open once

3) Finish setup inside Toast
   • Paste a read-only, team-scoped Vercel token
   • Pick a team and the projects to watch
   • Optionally enable notifications and Launch at Login
