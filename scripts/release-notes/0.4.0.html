What's New in 0.4.0

Anonymous analytics (opt-out)

Toast now sends anonymous usage data and crash reports to help fix bugs and improve the app. This is enabled by default.

  • We never collect your Vercel token, project names, or personal information
  • Turn off anytime in Settings → Privacy → Help improve Toast
  • Send feedback from the menu bar
  • Automatic crash reports with a one-tap prompt after unexpected quits
