Toast

A lightweight MacOS menu bar application to check your vercel deployments

Download for macOS

First-time install on macOS

Download the .dmg, open it, and drag Toast into Applications. On first launch, right-click the app and choose Open (required because the app is not notarized). Confirm when macOS asks.

Automatic updates are delivered via Sparkle once the app is installed.

Before opening the DMG, verify its SHA-256 checksum against SHA256SUMS.txt on the GitHub release: 

shasum -a 256 ~/Downloads/Toast-*.dmg
