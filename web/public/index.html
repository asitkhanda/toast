Vercel Status

A macOS menu bar app that shows live Vercel deployment status for your watched projects.

Download for macOS

First-time install on macOS

Download the .dmg, open it, and drag Vercel Status into Applications. On first launch, right-click the app and choose Open (required because the app is not notarized). Confirm when macOS asks.

Automatic updates are delivered via Sparkle once the app is installed.
