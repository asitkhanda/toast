A lightweight MacOS menu bar app to check your vercel deployments

Install with Homebrew 
brew tap asitkhanda/toast-tap
brew trust asitkhanda/toast-tap
brew install --cask toast-app

Install in three steps

Toast is Developer ID signed and Apple-notarized, so macOS should open it normally after you copy it to Applications.

1

Copy to Applications

Open the downloaded DMG and drag Toast into Applications. Don’t open it from the DMG itself.

2

Open Toast

In Applications, double-click Toast. Notarized builds open without Gatekeeper workarounds.

Fallback on older downloads: right-click Toast → Open → Open once.

3

Connect Vercel

Paste a read-only, team-scoped Vercel token, pick projects to watch, and optionally enable notifications and Launch at Login.

Need the steps again later? Open How to Open Toast.txt on the DMG.
